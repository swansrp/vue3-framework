# 框架常用组件说明

本文档介绍框架中常用的组件及其使用场景。**在开发新功能时，请优先使用框架提供的组件，避免重复造轮**。

## 布局组件

### ContentLayout - 左右分栏布局
**路径**: `src/framework/components/common/Content/ContentLayout.vue`

**功能**: 提供左右分栏布局，常用于树形导航 + 详情内容的场景

**使用场景**:
- 左侧树形菜单 + 右侧内容区域
- 侧边栏 + 主内容区域

**示例**:
```vue
<ContentLayout :width="280">
  <template #side>
    <!-- 左侧内容：树形菜单、导航等 -->
  </template>
  <template #content>
    <!-- 右侧内容：详情、表格等 -->
  </template>
</ContentLayout>
```

## 数据展示组件

### Portal - 通用表格门户
**路径**: `src/framework/components/common/Portal/index.vue`

**功能**: 功能强大的通用表格组件，支持增删改查、高级搜索、导入导出等

**核心特性**:
- 自动生成表格和表单
- 支持多种显示模式（列表、网格、树形）
- 内置增删改查操作
- 高级搜索和筛选
- 关联数据绑定
- 数据导入导出
- 自定义按钮和操作

**相关文档**: `PORTAL组件使用指南.md`

**使用场景**:
- 标准的 CRUD 页面
- 数据管理页面
- 需要复杂查询和筛选的列表页

### DictContent - 字典内容
**路径**: `src/framework/components/common/Content/DictContent.vue`

**功能**: 字典数据展示和管理

### GeneralDashboard - 通用仪表盘
**路径**: `src/framework/components/common/GeneralDashboard/index.vue`

**功能**: 通用的数据仪表盘组件，支持多维度数据展示和筛选

**核心特性**:
- 指标树选择
- 维度筛选
- 图表展示
- 数据统计

**使用场景**:
- 数据分析页面
- 统计报表
- 可视化大屏

## 表单组件

### DynamicFormItem - 动态表单项
**路径**: `src/framework/components/common/DragGrid/DynamicFormItem.vue`

**功能**: 可配置的动态表单项，根据配置自动生成表单字段

**使用场景**:
- 需要动态配置的表单
- 表单字段不固定的场景

### DragGrid - 可拖动布局
**路径**: `src/framework/components/common/DragGrid/`

**功能**: 可拖动调整的网格布局，支持表单项拖拽排序和布局调整

**核心组件**:
- `GridDraggableLayout.vue` - 可拖动布局容器
- `GridDraggableForm.vue` - 可拖动表单
- `GridDraggableFormModal.vue` - 可拖动表单弹窗

**使用场景**:
- 自定义表单布局
- 可配置的仪表盘
- 需要用户自定义布局的页面

## 开发建议

### 1. 优先使用框架组件
在开发新功能时，请先检查框架是否已有类似组件，避免重复开发。

### 2. 参考已有实现
如需扩展或自定义，建议参考框架组件的实现方式，保持代码风格一致。

### 3. 查阅配套文档
部分组件有详细的使用文档，使用前请先阅读相关文档。

### 4. 常用组合
- **左右分栏 + Portal**: `ContentLayout` + `Portal` 实现标准的数据管理页面
- **仪表盘页面**: `ContentLayout` + `GeneralDashboard` 实现数据分析页面
- **表单弹窗**: `Portal` 内置的 `PortalEditModal`
- **文件上传**: Portal 支持文件字段，自动集成 `UploadFile`
