# 开发备忘录

## Ant Design Tree 组件样式对齐问题

### 问题描述
当使用 `<a-tree>` 组件并自定义 `#title` 插槽时，容易出现图标与文本不对齐的问题。

### 解决方案

#### 1. 树节点容器使用 flex 布局
```less
.ant-tree-treenode {
  display: flex;
  align-items: center;  // 关键：让所有子元素垂直居中对齐
}
```

#### 2. 统一所有元素的行高
```less
// 内容包装器
.ant-tree-node-content-wrapper {
  line-height: 22px;
  vertical-align: middle;
}

// 自定义插槽容器
.tree-node-wrapper {
  line-height: 22px;
  vertical-align: middle;
}

// 文本内容
.tree-node-title,
.product-title {
  line-height: 22px;
  vertical-align: baseline;
}
```

#### 3. 图标垂直对齐
```less
// 切换器图标（展开/收起）
.ant-tree-switcher {
  display: inline-flex;
  align-items: center;
  vertical-align: middle;
  
  .ant-tree-switcher-line-icon {
    vertical-align: middle;
  }
}

// 节点图标
.ant-tree-iconEle {
  display: inline-flex;
  align-items: center;
  vertical-align: middle;
  
  svg {
    vertical-align: middle;
  }
}
```

#### 4. 自定义内容使用 inline-block
```less
.product-tree-node {
  display: inline-block;
  line-height: 22px;
  vertical-align: middle;
}
```

### 注意事项
- **关键点**：`.ant-tree-treenode` 必须使用 `display: flex` + `align-items: center`
- 所有文本和容器统一使用相同的 `line-height`（建议 22px）
- 图标元素使用 `inline-flex` + `vertical-align: middle`
- 避免使用多层 flex 嵌套，可能导致对齐问题
- 自定义插槽内容使用 `inline-block` 而不是 `flex`
