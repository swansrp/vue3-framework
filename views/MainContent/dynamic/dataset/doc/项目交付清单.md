## 🔧 使用方式

### 快速开始（3步）

1. **创建Dataset**
   ```
   打开页面 → 点击"新建" → 输入SQL → 预览 → 保存
   ```

2. **配置列信息**
   ```
   选择Dataset → 点击"列配置" → 调整列设置 → 保存
   ```

3. **查询数据**
   ```
   选择Dataset → 切换到"数据查询" → 填写条件 → 查询
   ```

详细使用说明请查看：[QUICKSTART.md](QUICKSTART.md)

## 📡 API对接

### 需要对接的API接口（11个）

#### Dataset配置管理（5个）
1. `POST /web/dataset/config/parse` - 解析SQL预览
2. `POST /web/dataset/config/replace` - 保存Dataset配置
3. `GET /web/dataset/config` - 获取Dataset配置
4. `POST /web/dataset/config/query` - 查询Dataset列表
5. `POST /web/dataset/config/delete` - 删除Dataset

#### 列配置管理（5个）
6. `GET /web/dataset/config/column/list` - 获取列配置
7. `POST /web/dataset/config/column/add` - 添加列
8. `POST /web/dataset/config/column/update` - 更新列
9. `POST /web/dataset/config/column/delete` - 删除列
10. `POST /web/dataset/config/column/update/order` - 更新列顺序

#### 数据查询（1个）
11. `POST /web/dataset/data/query` - 查询数据

详细API说明请查看：[API对接说明.md](API对接说明.md)
